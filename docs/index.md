隐形互联网协议（守护进程）
====================================

i2pd（I2P 守护进程）是 I2P 客户端的全功能 C++ 实现。

I2P（隐形互联网协议）是一个通用的匿名网络层。 
所有通过 I2P 的通信都是匿名且端到端加密的，参与者不会暴露其真实 IP 地址。 

I2P 客户端是一种用于构建和使用匿名 I2P 
网络的软件。这类网络通常用于匿名的点对点 
应用（文件共享、加密货币）以及匿名的客户端—服务器 
应用（网站、即时通讯、聊天服务器）。

I2P 让全世界的人们能够不受限制地交流和分享信息。

特性
--------

* 分布式匿名网络框架
* 端到端加密通信
* 占用小、依赖简单、性能快速
* 为安全应用开发者提供丰富的 API 集合
